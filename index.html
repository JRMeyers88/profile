AnkiJisho - Capstone Project

AnkiJisho is a Japanese English dictionary and study tool designed for language learners that want to discover and practice vocabulary. It allows a user to search for any word or phrase in either Japanese or English, save the results into user created folders, and then practice those words as flashcards. Utilized AngularJS, Firebase, Materialize, CSS
